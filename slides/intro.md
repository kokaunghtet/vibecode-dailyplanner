# Daily Planner

A personal task & calendar app for people juggling too much.

<!-- 20s -->

---

<!-- slide 2 -->

# Who it's for

Students and anyone juggling assignments, errands, and deadlines who:

- scatters tasks across sticky notes, apps, memory
- has no calendar view of what's coming up
- struggles to track daily progress

---

<!-- slide 3 -->

# What it does

- Add, edit, complete, delete tasks per day
- Calendar view — browse and plan by date
- Dark/light mode toggle
- Persistent, per-user data via Firebase Firestore + Auth

---

<!-- slide 4 -->

# Why it matters

- **One place** for all daily tasks — no scattered to-dos
- **Calendar view** — see what's coming, plan ahead
- **Private per-user data** — secure via Firebase Auth
- **Dark mode** — comfortable planning any time
